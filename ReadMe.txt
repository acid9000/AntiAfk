20260118: Converted and updated from 10-yr-old German addon "aafk".

To enable/disable in game:
1. Type /aafk

How it works:
1. Any time a player or NPC yells in your zone, this will send a message of "aafk" to the channel "LFT". 
2. Sending a chat message restarts the in-game AFK timer, so if you're querying the entire Auction House in Aux while physically in a city like Stormwind or Orgrimmar, you can now do it without getting kicked or needing to interact with your client at all.

Note: 
You do not need to have LFT turned on in your chat window(s)- it is the backend channel for many addons to share data (e.g. PizzaWorldBuffs and Aux).

Original Readme:

Deutsch:
Damit das Addon funktioniert müsst ihr es mit "/aafk" (ohne ") anschalten. Mit dem gleichen befehl lässt es sich auf wieder beenden.
Außerdem muss bei Interface Optionen "Auto Clear AFK" angeschaltet sein.

Das Addon whispert, wenn es eine Nachricht im Say/Raid/Party/Officer Chat erhält, den Spieler selber an und bringt ihn so ausm afk.


English:
To get the Addon working you have to type "/aafk" (without ") ingame. You can use the same command to turn it off again.
Addiationally you have to turn on "Auto Clear AFK" in your Interfaceoptions.

If you recieve a message from Say/Raid/Party/Officer-Chat the Addon whispers to yourself. This causes you to get out of AFk-mode.
